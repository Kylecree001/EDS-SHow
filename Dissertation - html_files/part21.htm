IV: Docker

Images have now been taken care of, and therefore is time to start constructing and training the model. The Docker [39] application will be used to aid in this process. Docker is an open-source platform that allows the user to create self-contained containers for applications. The containerising application provides many benefits, such as easy deployment, easy management and installation of packages, no outside variables affecting the data, and reproducibility. For this project, Docker was utilised to install a standard version of all libraries and packages needed and make the results reproducible as the environment is the same.


The model used is from the image segmentation keras version 2.4.1_0.3.0 framework [41] and was installed in a docker container using the Docker image. Once installed, a container environment is started and used throughout the training and testing process.
