Bottle recycling machine with machine learning

Development of Bottle Recycling Machine Using Machine Learning Algorithm [13] proposes the use of a bottle recycling machine, with a machine-learning algorithm to identify the material used. The proposed machine works as a backward vending machine, wherein a user inputs an empty bottle to the machine. The machine then uses either a sensor to detect if it is aluminum cans or

otherwise machine learning model to predict the material. The bottle is then sorted into a bin determined by the classification.


To ensure that the public use this machine it offers a coupon for every bottle that is input and recyclable. The design of the machine has an inbuilt printer to output the coupon for the user. Doing this will give people more incentive to recycle their materials as well as others that have been left on the ground.


Another part of this idea comes around the point of needing to empty the bins within the machine. To do this the machine is set to notify the owner through an SMS message. This will mean that the recycling machine should never be overflowing, helping keep away both problems created by too much trash around as well as making sure the machine is always available for users.
