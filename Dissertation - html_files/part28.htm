Chapter 7: Conclusion


This dissertation brought forward the idea of using machine learning to detect rubbish that contaminated recycling, leading to valuable recycling ending up in the landfill. The project followed a machine learning workflow where first, the images were collected and labelled. Next, those images were used to train different models, testing the model and recording the results. Unfortunately, due to circumstances, the wanted images, which were initially planned to be from a recycling truck, were unobtainable, and therefore other images needed to be used. This caused the training and test images to differ, affecting the results.


Although the accuracy was not the highest, this dissertation does produce some valuable information about the topic. First, the images required for this project do need to be collected from a real-world environment to produce better results. However, results aside, this dissertation presents a relevant way to label, train and test this idea. Considering how different the training and test data sets were, the results show that image segmentation is a suitable technique that should produce good results with the right tools.
