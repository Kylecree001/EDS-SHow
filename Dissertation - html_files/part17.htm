Chapter 5: Implementation


Once the images are collected, the next step is to set up the model. Image segmentation is the machine learning technique used in this project, in which images are partitioned into smaller sections that are then individually labelled for identification. Next, the collected images are resized and labelled, split into train, validation, and test sets, which are subsequently used to train the model and produce results. Many different technologies were used, such as photoshop, ADAMS [42], Docker [39], Tensorflow [44], and the image-segmentation-keras framework [41].
