I: Introduction

An idea is proposed to use machine learning and the developments in computer vision to create a program that could assist in reducing the amount of recycling ending up in landfill. In the past, research has been carried out in this area, such as creating a bin that sorts the items into the correct categories, a robot that detects, picks up the rubbish, and sorts it, mobile applications, and bottle recycling machines. All of these ideas provide relevance in both knowing the problem has some possibility of being solved and seeing how the papers solved specific problems.


This dissertation section reviews current research on technology around machine learning and how it can help with the current contamination levels in recycling through detection software.
